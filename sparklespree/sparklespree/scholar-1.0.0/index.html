PHP

Hypertext Preprocessor atau PHP adalah bahasa penulisan skrip open-source yang banyak digunakan dalam pemrograman atau pengembangan website (web development). PHP biasanya digunakan bersama JavaScript dan Python.

HTML

HTML, singkatan dari HyperText Markup Language, adalah bahasa standar yang digunakan untuk membuat dan merancang halaman web. HTML menggunakan serangkaian elemen dan tag untuk menyusun konten di web, seperti teks, gambar, tautan, dan berbagai media lainnya.

MySQL

MySQL adalah sistem manajemen basis data relasional open-source yang menggunakan SQL (Structured Query Language) untuk mengelola dan memanipulasi data. MySQL sering digunakan dalam pengembangan web dan aplikasi, Sistem ini mendukung berbagai platform .
